### Setup Instructions for frontend team: 
 1- git clone my repo url 
 2- then go in the backend folder and in the terminal run: "npm i" for dependencies installation 
 3- before running backend server run main.cpp file: go in the core-engine folder then run: "g++ main.cpp -o engine.exe" , after running this command "engine.exe" file will create in the same folder , then copy this file and paste in the backend folder now you have done all the required steps . 
 4- go in the backend folder , run: "node server.js" command Backend is ready Now !!!!
